# DentistAppointment

Project setup:
1. Build the project
2. Delete the project.assets.json
3. Clean Solution
4. Go to Tools -> NuGet Package Manager -> Package Manager Console
5. Execute 'dotnet-restore'
6. Build Solution
7. Updata-Database
